## Checklist

### Legales y administrativos
- [ ] Contrato marco (MSA) firmado
- [ ] NDA firmado
- [ ] SOW firmado
- [ ] Pago inicial recibido y confirmado

### Contactos y accesos
- [ ] Contacto principal del cliente definido (nombre, email, teléfono)
- [ ] Stakeholders del cliente identificados (quiénes participarán)
- [ ] Canal de comunicación acordado (email, Slack, Teams, WhatsApp)
- [ ] Carpeta compartida creada (Google Drive / repositorio)

### Insumos del cliente
- [ ] Documentos institucionales recibidos (estrategia, reportes, presentaciones)
- [ ] Contexto del problema / decisión documentado
- [ ] Restricciones internas explicitadas (confidencialidad, plazos, política)
- [ ] Horizonte temporal del análisis definido

### Alineamiento
- [ ] Objetivo del proyecto validado con el cliente
- [ ] Alcance (incluye / no incluye) validado
- [ ] Cronograma validado y fechas clave acordadas
- [ ] Kickoff agendado
- [ ] Formato de entregable acordado (memo, dashboard, workshop)
