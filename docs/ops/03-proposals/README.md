## Estructura de la propuesta (según mentoría)

1. Portada (logo, proyecto, cliente, fecha, confidencial)
2. Resumen ejecutivo (problema, propuesta, resultado, plazo, inversión)
3. Contexto del cliente (situación, decisión crítica, por qué ahora)
4. Objetivo del proyecto (1 general + 3-5 específicos)
5. Alcance (incluye / no incluye)
6. Metodología (lectura → ontología → simulación → reporte/workshop)
7. Entregables (memo, dashboard, escenarios, recomendaciones, workshop)
8. Cronograma (semana a semana)
9. Inversión (precio, impuestos, forma de pago, vigencia)
10. Supuestos (acceso, disponibilidad, feedback, confidencialidad)
11. Próximo paso (firma, NDA, kickoff, pago)
