# Docs

This folder holds the public-facing documentation that makes the repository easier to understand from the outside.

## Public docs

- `brand-guidelines.md`
  Design and identity rules for the Dasein Foresight brand system.
- `design-principles.md`
  Short explanation of the product, narrative, and interface choices behind the site.
- `research-signal-count-methodology.md`
  Research note that informs one part of the venture's signal framing.

## What belongs here

Add documentation here when it helps an external visitor answer one of these questions:

- What is this project?
- Why is it shaped this way?
- What claims are grounded by evidence?
- What is real versus still emerging?

## What does not belong here

- throwaway experiments
- local tooling notes for one AI assistant
- uncurated reference dumps
- working files that only make sense to the repo owner

Those are better kept outside the public-facing documentation path.
