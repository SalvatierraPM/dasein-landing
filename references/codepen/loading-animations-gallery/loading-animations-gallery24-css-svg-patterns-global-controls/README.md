# Loading Animations Gallery - 24 CSS + SVG Patterns (Global Controls)

A Pen created on CodePen.

Original URL: [https://codepen.io/ash1198/pen/XJXVqMK](https://codepen.io/ash1198/pen/XJXVqMK).

A curated set of 24 production-ready loaders you can drop into any UI.
Includes: Spinner, Dual Ring, Dots (Pulse/Bounce), Indeterminate Bar, Striped Bar, Ripple, Pulse, Ring Dash (SVG), Infinity path (JS-driven for cross-browser stability), Hourglass, Radar, Equalizer, Typing, Tiles 3×3, Coil, Comet, Arc Spinner (conic), Wave, Flip 3D, Triangle Spin, Orbit, Skeleton Line/Card (Shimmer).

Features
- Global controls: Size, Speed, Color, Pause, Theme (Dark/Light)
- Respects prefers-reduced-motion (except Infinity which is JS-controlled for reliability)
- Clean, readable CSS keyframes + minimal JS, no libraries
- Accessible structure, responsive cards, copy-paste friendly classes

How to use
- Tweak sliders to preview; copy the .loader.* block you need.
- Works in modern browsers; Infinity uses requestAnimationFrame for smooth dash travel.
