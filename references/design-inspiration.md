# Design Inspiration Reference Library

Use these sources BEFORE writing frontend code. Search for components, patterns, and visual direction.

## MCP Tools (use first)
- **@21st-dev/magic** — Real component library with code. Search: "hero section dark", "pricing bento", "dashboard glassmorphism"
- **shadcn/ui MCP** (via ui-styling skill) — Base components to customize, never use as-is

## Component Libraries to Reference
When using WebFetch or browsing for inspiration:

### Premium UI References
- `ui.aceternity.com` — Animated components (spotlight, aurora, meteors, 3D cards)
- `magicui.design` — Animated components with framer-motion
- `ui.shadcn.com` — Base primitives (customize heavily)
- `hover.dev` — Animated UI components
- `animata.design` — Copy-paste animated components
- `luxe.guhrodrigues.com` — Luxurious animated components

### Design Galleries (visual direction)
- Awwwards — award-winning web design
- Godly.website — curated landing pages
- Landingfolio — landing page inspiration
- Mobbin — mobile UI patterns
- Dribbble — concept exploration (filter by "web design")

### Pattern-Specific
- `www.dark.design` — dark mode done right
- `neubrutalism.dev` — neo-brutalist components
- `glassmorphism.com` — glass effects reference
- `bennettfeely.com/clippy` — CSS clip-path shapes
- `cubic-bezier.com` — custom easing curves
- `coolors.co` — palette generation (convert to oklch)
