Entangled - Quantum Particle Visualization

Recreation of Bjørn Staal's generative art piece exploring quantum entanglement through synchronized particle systems.
Original: nonfigurativ.com/projects/entangled

Physics System

  • Spring-Mass Network: Each particle tethered to home position with LFO-modulated springs
  • Curl Noise Fields: 3D organic flow for wave-like motion
  • LFO Modulation: Breathing rhythm at ~2.5 second cycles (heartbeat-like)
  • Stochastic Coupling: 10% probability impulses for entanglement synchronization
  • Particle Count: 12,000 particles in Fibonacci sphere distribution

Visual Effects

  • Intrinsic Glow: Exponential falloff exp(-12 * d²) in shader (no post-processing)
  • Energy Colors: Cyan (#00faff) when stable → Gold (#ffb873) when energized
  • Depth Fog: Subtle violet atmospheric tinting
  • Additive Blending: Particles blend with GL_SRC_ALPHA, GL_ONE

Interaction Controls

  • Mouse Drag: Rotate the particle cloud view
  • Mouse Hover: Inject turbulent energy (creates warm color bursts)
  • Scroll Wheel: Adjust camera distance (zoom in/out)
  • Pairing: Opens synchronized partner window with asymmetric coupling

Technical Implementation

  • WebGL 2.0: Hardware-accelerated GPU rendering
  • GLSL Shaders: Custom vertex/fragment shaders with noise functions
  • Zero Dependencies: Pure JavaScript and WebGL
  • Performance: 60 FPS target with 12K particles
Note: For best experience, open the main window and then the paired window side-by-side. The paired window will show synchronized particle motion with phase-shifted coupling, demonstrating the "quantum entanglement" effect where changes in one system influence the other non-linearly.
